# Migdal Financial Services – Interactive Dashboard

Interactive Streamlit dashboard for **Migdal Financial Services** client analytics, churn prediction, and data correction.

## Features
- **Part A – Process Mapping**: Visual Sankey flow-charts of the current and proposed lead-handling process.
- **Part B – Management Dashboard**: KPI cards, trend charts, city/service/status drill-downs, satisfaction analysis.
- **Part B (cont.) – Data Correction & Export**: Re-assign `client_id` by join-date order and export corrected Excel.
- **Part C – Insights & Prediction**: Churn analysis, risk profiling, XGBoost model rationale, macro-data discussion.

## Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
